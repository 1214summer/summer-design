# Summer-design
一个供自己使用的从0开始的react组件库
## 使用技术栈
- 包管理器：npm
- 框架： react
- css预处理语言：sass
- 组件库参考：antd
- 打包：vite
- 项目文档: storybook
- 代码规范工具链：eslint + prittier + stylelint
- git提交规范链： husky + commitlint + lint-staged
